# jsf-jdbc-crud
jsf + jdbc CRUD webapp

Webapp using jdbc (POSTGRESQL, DataBaseName=Example, user=postgres, password=postgres) + javax.faces + WEBServer WildFly 20.0.0.Final + bootstrap (awesome buttons)
implements CRUD operations.
